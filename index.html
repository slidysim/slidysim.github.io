Does not work?

Important notes (to avoid file permission and other issues):

- Avoid placing files in "Program Files", the C:\ root directory, or other protected folders
- Extract all files to an easily accessible location (like a folder on your desktop)
- Do not run any program directly from the archive — always extract first
- Do not run the PBs Manager via the Windows search bar
- Open it from its folder directly (or create a shortcut for it)
- The PBs Manager is named "SlidysimRecordsManager1.3.exe"
- Make sure it's in the same folder as your other Slidysim files
- Do not confuse it with "slidysim.exe", it does not replace it
- Do not manually modify Slidysim files

What is this?

This is a fully independent sliding puzzle leaderboard for competitive Slidysim players. It supports exe uploads through PBs Manager and web Slidysim scores through the web archive views.

How do I get access?

You do not need an account to view the live or archive leaderboard pages.

Web players should make an account at official web Slidysim. Web scores are available through the web archive on this site.

Exe players who want to upload records with PBs Manager need a registration token. Request one by contacting dphdmn or vovker via Discord, or dphdmn through Telegram. (You can also use Google Form).

Token registration helps prevent impersonation, multi-accounting, and duplicated results.

Exe players can register with the form on this page using a unique token. The token is linked to your username for PBs Manager uploads.

Still confused? Join our Discord Server.

How do I upload records?

Download the Slidysim Records Manager tool from GitHub. It’s a Windows version. For Linux version, there is working experimental release by MOKA (feel free to ask about it in our discord).

Place the executable in the same folder as Slidysim and run it. It will automatically launch Slidysim and track your records in real-time, it will close when you exit Slidysim for a seamless experience. Manual updates are available if needed (you don't even have to run it when playing if you don't want to, just run it occasionally to upload records). Initial record uploads might take 1-2 minutes, but subsequent uploads should take only a few seconds. Avoid modifying Slidysim files manually and contact us if you encounter any issues.

Why?

At the end of September 2024, the creator of Slidysim restricted access to the leaderboard due to personal conflicts with a certain top player. Initially, some results were removed, making the leaderboard unreliable. Later, the API for leaderboard access broke, disrupting custom ranking sites like the Better Leaderboard and Power Rankings.

Soon after, the leaderboard data became corrupted with random noise, rendering it unusable. Eventually, the official leaderboard access was completely blocked, causing the game to crash when attempting to view it. The creator has attempted to shift the blame for this disruption on the player he had conflict with, but we believe it is unjustified. Our goal is to keep the Slidysim rankings open for everyone, providing a reliable and independent alternative to the original leaderboard.

This section stays here for historical context. Despite those old issues, the website now fully supports both exe and web Slidysim platforms.
